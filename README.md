# potts-TFG-Fis2026
Simulaciones Monte Carlo del modelo de Potts de q estados aplicado a mercados financieros. Implementa un campo de pánico global η(t) que desplaza la temperatura crítica y colapsa burbujas especulativas. Reproduce cuatro hechos estilizados: colas pesadas, agrupamiento de volatilidad, ausencia de autocorrelación lineal y asimetría ganancia-pérdida.
